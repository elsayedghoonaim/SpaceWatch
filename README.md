# SpaceWatch: Real-time Orbit Tracking & Collision Avoidance System

SpaceWatch is a comprehensive, high-fidelity orbital tracking and collision avoidance system. It is designed for satellite operators, researchers, and space agencies to enhance space safety and sustainability. The project provides accurate, real-time predictions of satellite and debris trajectories to minimize collision risks and promote safer space operations.

## Key Features

* **Real-time Orbital Visualization**: A dynamic 3D interface to visualize the real-time position and trajectory of satellites and debris in orbit around the Earth.
* **ML-Enhanced Predictions**: The system uses the standard SGP4 propagation model, significantly enhanced with proprietary machine learning algorithms. This hybrid approach improves accuracy, especially for objects in complex orbital regimes or during high solar activity.
* **Conjunction Analysis and Maneuver Planning**: Identifies high-risk conjunction events, providing details like Time of Closest Approach (TCA), miss distance, and probability of collision. It also recommends avoidance maneuvers.
* **Space Weather Monitoring**: A dashboard to monitor solar and geomagnetic activity, including solar wind, solar flares, sunspot cycles, and auroral power, which can affect satellite operations.
* **Ground Coverage Prediction**: A map-based tool to predict a satellite's ground track and coverage area at a specified time.
* **Detailed Documentation**: The project includes documentation on the SGP4 model, machine learning corrections, data sources, and collision risk assessment methodologies to ensure transparency.
